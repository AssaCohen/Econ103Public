Lecture Slides

Lecture slides will be posted here shortly after each lecture. If there is ever an unforseen delay in posting, you can also access my slides via github. If you choose to download slides from github rather than the course website, please be aware that I often make changes shortly before class. Since github provides an exact copy of whatever is on my computer at any given moment, you may get a preliminary rather than final version of the slides if you access them on github before the lecture takes place.
